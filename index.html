Project: Rock Paper Scissors

This project is a web-based implementation of the classic game "Rock Paper Scissors," where users can play against a computer by choosing between rock, paper, and scissors options. The user's and computer's scores are displayed, along with the choices made in each round.

Project: Landing Page

This project is a cat-themed website that showcases a cute cat and provides playful interactions. It includes sections for social media links, information about the cat's behavior, quotes, and a call-to-action button for feeding the cat. The website's design emphasizes a cat-loving atmosphere, complete with a cat logo and adorable images. The footer contains a dynamic copyright notice.

Project: Recipes

This webpage titled "Pinoy Style" presents a collection of Filipino recipes. It features an image of a bamboo cookhouse and provides links to popular Filipino dishes such as Chicken Arroz Caldo, Traditional Filipino Lumpia, and Famous Chicken Adobo. Users can navigate through the list of recipes and also find a link to return to the top of the page.
